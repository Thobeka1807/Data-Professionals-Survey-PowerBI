This folder contains screenshots and visuals from the Power BI dashboard.

- dashboard_overview: Full dashboard view.
- salary_by_job: Stacked bar chart of average salary by job title.
- language_by_job: Stacked column chart of programming language preferences.
- treemap_locations: Tree map showing where survey takers live.
- gauges: Gauges for work/life balance and salary satisfaction.
- donut_difficulty: Donut chart showing difficulty of entering the industry.
